# 15-Puzzle game

![Alt text](https://nasiadka.pl/projects/15puzzle/15puzzle.png)


The 15-puzzle game is a sliding puzzle having 15 square tiles numbered 1â€“15 in a frame that is 4 tiles high and 4 tiles wide, leaving one unoccupied tile position. Tiles in the same row or column of the open position can be moved by sliding them horizontally or vertically, respectively. The goal of the puzzle is to place the tiles in numerical order.
